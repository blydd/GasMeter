# 油耗记（GasMeter）— 产品需求文档（PRD）

> **一句话定位**：油耗记是一款面向个人车主的极简油耗记录工具，以"3 秒完成一笔加油记录"为核心体验，全部数据本地存储，支持 CSV 导入导出与多车辆管理。

---

## 1. 产品目标

| # | 目标 | 衡量标准 |
|---|------|---------|
| G1 | **极速记录**：用户从打开 App 到完成一笔加油记录 ≤ 3 步操作 | 核心路径点击次数 |
| G2 | **离线优先**：所有数据 100% 本地存储，无需联网、无需注册 | 无网络权限请求 |
| G3 | **多车管理**：同一用户可管理多辆汽车/摩托车，切换便捷 | 支持车辆数 ≥ 10 |
| G4 | **数据可迁移**：CSV 导入导出保证数据不丢失、不锁死 | 导入/导出成功率 100% |
| G5 | **可视化洞察**：月度油耗趋势、百公里油耗计算等基础统计 | 至少 3 种图表类型 |

---

## 2. 目标用户画像

| 画像 | 特征 | 核心诉求 |
|------|------|---------|
| **通勤车主** | 每天开车上下班，固定油站加油，关心每月油费 | 快速记录、月度汇总 |
| **家庭多车用户** | 家里 2-3 辆车，需要分别记录、对比 | 多车切换、车辆纬度统计 |
| **摩托车玩家** | 1 辆或多辆摩托车，关注油耗变化感知车况 | 支持摩托车、油耗趋势异常提醒 |

## 4. 需求池

### P0 — 必须有（MVP）

| ID | 功能 | 描述 |
|----|------|------|
| F01 | 快捷加油记录 | 首页显著位置「记一笔」按钮，弹出半屏表单（里程、油量、金额、是否加满），确认即保存 |
| F02 | 首页仪表盘 | 当前车辆卡片：最近油耗、本月总油费、最近记录摘要 |
| F03 | 车辆管理 | 添加/编辑/删除车辆，字段：名称、类型(汽车/摩托车)、品牌、型号、车牌、燃油标号 |
| F04 | 车辆切换 | 首页顶部车辆选择器，左右滑动或下拉切换当前车辆 |
| F05 | 记录列表 | 当前车辆的所有加油记录，按时间倒序，每条显示日期/里程/油量/金额/油耗 |
| F06 | 油耗自动计算 | 每次记录后自动计算：本次百公里油耗 = 油量 / (本次里程 - 上次里程) × 100（仅加满时计算） |
| F07 | SwiftData 本地存储 | 所有数据通过 SwiftData 持久化，无网络请求 |

### P1 — 应该有（V1.1）

| ID | 功能 | 描述 |
|----|------|------|
| F08 | 月度油耗趋势图 | 折线图：每月平均百公里油耗变化 |
| F09 | 月度油费柱状图 | 柱状图：每月总油费对比 |
| F10 | CSV 导出 | 设置页 → 导出全部数据为 CSV，可分享/存储到文件 |
| F11 | CSV 导入 | 设置页 → 从文件选择 CSV 导入，字段映射校验 |
| F12 | 记录编辑/删除 | 左滑删除、点击编辑任意加油记录 |
| F13 | 车辆编辑 | 修改车辆信息（名称、车牌、燃油标号等） |
| F14 | 统计页 | 独立 Tab：当前车辆的总里程、总油费、平均油耗 |

### P2 — 可以有（V1.2+）

| ID | 功能 | 描述 |
|----|------|------|
| F15 | 未加满智能合并 | 标记"未加满"的记录，下次加满时自动合并计算准确油耗 |
| F16 | 加油提醒 | 基于历史加油频率，低油量提醒 |
| F17 | 油耗异常预警 | 油耗突然偏高时标红提示（可能是车况问题） |
| F18 | 深色模式 | 跟随系统 / 手动切换 |
| F19 | iCloud 备份 | 可选开启 iCloud 同步（仍无服务端） |

## 5. 数据模型概览

### Vehicle（车辆）

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| id | UUID | ✅ | 主键 |
| name | String | ✅ | 车辆昵称，如"小白" |
| type | Enum | ✅ | car / motorcycle |
| brand | String | | 品牌，如"丰田" |
| model | String | | 型号，如"卡罗拉" |
| year | Int | | 年份 |
| licensePlate | String | | 车牌号 |
| fuelType | String | | 常用燃油标号，如"92#"、"95#" |
| createdAt | Date | ✅ | 添加时间 |

### FuelRecord（加油记录）

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| id | UUID | ✅ | 主键 |
| vehicle | Relationship | ✅ | 关联车辆 |
| date | Date | ✅ | 加油日期，默认当天 |
| odometer | Double | ✅ | 当前里程表读数 (km) |
| fuelAmount | Double | ✅ | 加油量 (L) |
| pricePerLiter | Double | ✅ | 单价 (元/L) |
| totalCost | Double | ✅ | 总金额（自动计算：fuelAmount × pricePerLiter） |
| isFullTank | Bool | ✅ | 是否加满，默认 true |
| fuelType | String | | 本次燃油标号 |
| notes | String | | 备注 |
| createdAt | Date | ✅ | 记录创建时间 |

> **关系**：Vehicle 1 ←→ N FuelRecord（一对多，级联删除）

## 6. UI 设计稿

### 整体结构
```
┌─────────────────────────────────┐
│  ⛽ 油耗记                       │
│    [首页]  [统计]  [设置]         │  ← TabBar
└─────────────────────────────────┘
```

### 首页（快捷记录 + 仪表盘）
- 顶部车辆选择器（左右滑动切换）
- 仪表盘卡片：最近油耗、最近油价、本月油费、总里程
- 主 CTA「记一笔」按钮
- 最近记录列表

### 记录弹窗（半屏 Sheet）
- 日期（默认今天）
- 里程表读数 (km)
- 加油量 (L)
- 单价 (¥/L)
- 总金额（自动计算）
- 是否加满（默认是）
- 燃油标号
- 备注

### 统计页
- 月度油耗趋势折线图
- 月度油费柱状图
- 累计统计：总里程、总油费、总油量、平均油耗

### 设置页
- 用户信息
- 数据管理：导出/导入/清空
- 车辆管理入口
- 关于

## 8. 待确认问题

| # | 问题 | 建议默认 |
|---|------|---------|
| Q1 | 首页「记一笔」→ 半屏 Sheet 还是 push 新页面？ | 半屏 Sheet |
